# ✍️ AI-Powered Blog Application | PostCraft (2025)

PostCraft is a full-stack **AI-powered blogging platform** built using the **MERN stack**, designed to simplify content creation and blog management using generative AI.

---

## 🚀 Features

- 🔐 **JWT-Based Authentication**
  - Secure login and signup for users and admins
  - Role-based access control (Admin & User)

- 🤖 **AI Content Generation**
  - Integrated **Google Gemini API** for AI-driven blog content creation
  - Helps users generate high-quality blog drafts efficiently

- 🖼️ **Image Optimization**
  - Used **ImageKit** for optimized image storage and fast delivery
  - Improved website performance and load speed

- 🎨 **Modern Responsive UI**
  - Built with **React.js** and **Tailwind CSS**
  - Fully responsive design across devices
  - Real-time **Toaster notifications** for user actions

- 🔍 **Blog Management System**
  - Search and filter blog posts
  - CRUD operations for blog posts
  - Admin dashboard for managing users and content

---
